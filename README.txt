ΕΝΤΟΛΗ ΜΕΤΑΓΛΩΤΤΙΣΗΣ:
			make
ΕΝΤΟΛΗ ΕΚΤΕΛΕΣΗΣ:
			exe | less

γιατί βγαίνουν πολλά μηνύματα εκτυπώσεων και ο scroll buffer δεν είναι αρκετός ώστε
να "φαίνονται" όλα. Για πιο ελεγχόμενα μηνύματα, οι print κάθε κλάσης έχουν μπει σε σχόλια. 
Εάν θελήσετε να δείτε τις πληροφορίες κάθε σωματιδίου, αυτό είναι εφικτό, βγάζοντας τα σχόλια.
Επίσης, ο αριθμός των ατόμων που δημιουργούνται μπορεί να αυξηθεί ή να μειωθεί στη main, αλλάζοντας
το φράγμα.
